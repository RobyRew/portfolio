A typing-practice app where the passages you type are Bible verses. Born from "wouldn't it be nice if the things I type to get faster were also things I want to read more of?"

Four timing modes (15 / 30 / 60 / 120 seconds), live WPM and accuracy, multi-language passages, and a stripped-down UI that disappears while you're typing. Lexend font for reading-optimization. Dockerized + on GitHub Pages.
